BUILDING THIS IMAGE:
--------------------
Rebuilding procedure (if you want to modify further):

1. Extract Fragment 0:
   dd if=OrangeFox-STABILIZED-FIXED-V26.img of=fragment0.bin bs=1 count=4096

2. Prepare ramdisk:
   - Copy original V26 ramdisk files to directory
   - Add/modify recovery.fstab
   - Add/modify init scripts
   - Modify drivers as needed

3. Create CPIO:
   cd ramdisk_directory && find . | cpio -o -H newc > ramdisk.cpio

4. Assemble:
   cat fragment0.bin ramdisk.cpio > final_image.img

5. Verify Fragment 0:
   dd if=final_image.img of=frag0_verify.bin bs=1 count=4096
   md5sum frag0_verify.bin  # Should match: ea32a088bf262f20b8ccac32ab5da3e8
